  <h1>Raw Loader</h1>
  <p>A webpack loader to import files as a raw `{String}`</p>
</div>

<h2 align="center">Install</h2>

```bash
npm i -D raw-loader
```

<h2 align="center">Usage</h2>

**webpack.config.js**
```js
module.exports = {
  module: {
    rules: [
      {
        test: /\.txt$/,
        use: 'raw-loader'
      }
    ]
  }
}
```

```js
import txt from './file.txt';
```
